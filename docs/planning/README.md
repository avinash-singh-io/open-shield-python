# Project Planning

This directory contains all planning-related documentation.

## Structure
- **phases/**: Detailed implementation plans for each project phase.
- **decisions/**: Architecture Decision Records (ADRs).
- **tasks.md**: Current active task tracking.
- **backlog.md**: Future ideas and unprioritized work.
- **tech-debt.md**: Tracked technical debt and cleanup items.
- **roadmap.md**: High-level project timeline and milestones.
- **specs.md**: Detailed product specifications and requirements.
